Solution for Avazu Click-Through rate prediction


System Requirement
------------------
- 64-bit Mac OS X
- Python 2.7
- g++
- pypy
- sklearn
- at least 8GB memory and 40GB disk space

To reproduce our submission:
-------------------
- Download tha data("train" and "test") to this folder.
- Change directory to script:
	cd script
- Run the code:
	./run.sh
